1. Notación general

Un sistema está compuesto por un conjunto finito de contenedores estructurales, llamados nodos.

Cada nodo posee:

  • una carga actual (L)
  • una capacidad máxima (C)
  • una pertenencia a un grupo conservativo

La carga total de cada grupo se conserva en todo momento.

2. Holgura total (H)

La holgura total representa la cantidad agregada de capacidad aún no utilizada dentro de un sistema o grupo.

Definición conceptual:

Es la diferencia entre la capacidad máxima y la carga actual, agregada sobre todos los nodos activos.

Interpretación:

  • Mide cuánto espacio libre existe en el sistema
  • No considera si ese espacio es accesible o utilizable

Un valor H > 0 no implica que el sistema esté sano.

3. Holgura efectiva (Heff)

La holgura efectiva es la porción de la holgura total que puede utilizarse realmente para redistribuir carga.

Definición conceptual:

Es la holgura total ponderada por la accesibilidad estructural de cada nodo.

Factores que afectan Heff:

  • Conectividad entre nodos
  • Restricciones de flujo
  • Fricción estructural
  • Direccionalidad y válvulas

Cuando Heff = 0, el sistema ya no puede redistribuir presión, aunque exista holgura total disponible.

4. Entropía estructural (S)

La entropía estructural mide el desbalance interno en la distribución de carga dentro de un sistema.

Definición conceptual:

Es una medida de dispersión relativa de la carga respecto a la capacidad de cada nodo.

Interpretación:

  • Valores bajos indican balance interno
  • Valores altos indican concentración o tensión desigual

Una entropía elevada reduce la capacidad del sistema para absorber nuevas perturbaciones.

5. Estados derivados

A partir de las métricas fundamentales, el sistema reconoce estados operativos claros:

  • Vivo
    Heff > 0
  • Zombi
    H > 0 y Heff = 0
  • Colapsado
    No existe redistribución viable

6. Relación entre métricas

Las métricas no son independientes entre sí.

  • H define el límite superior de resiliencia
  • Heff define la resiliencia real
  • S modula la velocidad de degradación

El comparador estructural evalúa escenarios observando la evolución temporal de estas magnitudes.

7. Alcance y límites

Estas métricas describen propiedades estructurales, no semánticas.

No evalúan intención, significado ni valor externo. Evalúan únicamente viabilidad estructural.

Un sistema no colapsa cuando falla. Colapsa cuando ya no puede redistribuir presión.
