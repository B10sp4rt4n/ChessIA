Propósito del repositorio

Este repositorio contiene la versión pública del SHE Core, incluyendo su definición conceptual, métricas fundamentales y modos de aplicación.

El objetivo es permitir revisión técnica, discusión estructural y validación conceptual.

Qué incluye

  • Definición formal de métricas (H, Heff, S)
  • Comparador estructural v4.2
  • Modos de aplicación (grafo, ajedrez, escenarios)
  • Demos ejecutables con lógica simplificada

Qué no incluye

  • El motor completo de cálculo
  • Heurísticas de optimización
  • IA de decisión o predicción
  • Implementaciones productivas

Estado del proyecto

Esta versión corresponde a un core estable orientado a documentación técnica y demos.

La implementación completa del motor se mantiene fuera de este repositorio.

Audiencia esperada

Este repositorio está dirigido a:

  • Arquitectos de sistemas
  • Ingenieros de software
  • Investigadores en sistemas complejos
  • Revisores técnicos

Un sistema no colapsa cuando falla. Colapsa cuando ya no puede redistribuir presión.
